# FreeNetClamAV
The world’s first open-source enterprise-grade network antivirus, deeply customized on ClamAV engine. It fully supports Windows and mainstream Linux distributions including openEuler, Anolis OS, UOS, Kylin. Equipped with asset management, it provides enterprise-class endpoint security protection for users.
# 福瑞网络防病毒软件
全球首款开源企业级网络防病毒软件，基于 ClamAV 引擎深度定制。全面支持 Windows 及 openEuler、Anolis OS、UOS、麒麟等主流 Linux 发行版，内置资产管理功能，为用户提供企业级终端安全防护。
